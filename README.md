# 🏚️ БУНКЕР — Игра на выживание

## Стек
- **Frontend**: React + Vite + Framer Motion + Zustand
- **Backend**: Node.js + Express + Socket.io

## Запуск

### 1. Установить зависимости

```bash
# Backend
cd server
npm install

# Frontend  
cd ../client
npm install
```

### 2. Запустить backend
```bash
cd server
npm run dev
# Сервер запустится на http://localhost:3001
```

### 3. Запустить frontend
```bash
cd client
npm run dev
# Приложение откроется на http://localhost:5173
```

## Правила игры

### Фазы обсуждения (1–7)
- **Фаза 1**: Все раскрывают только профессию (обязательно)
- **Фазы 2–7**: Каждый раскрывает любую характеристику в свой ход

### Голосования
- **После фазы 3**: Первое голосование (2 мин → переголос 1 мин при ничьей)
- **После фазы 7**: Второе голосование

### Финал
- Все карточки всех игроков открываются
- Генерируется уникальный финальный текст с учётом катастрофы и персонажей
